# roadmap

## phase 1 — foundation

goal: ship the wizard shell and 2-3 hero assets to prove the architecture.

- [ ] wizard ui scaffold (category picker -> configure -> preview -> generate)
- [ ] rendering mode system (doodle / general / textured)
- [ ] doodle sub-toggle (filled vs lineart)
- [ ] texture library (embedded patterns for textured mode)
- [ ] color system (paper, ink, accent, aging tint)
- [ ] hero asset: polaroid
  - [ ] plain / clipped / pinned toggle
  - [ ] custom photo area
  - [ ] caption text field
- [ ] hero asset: wax seal
  - [ ] seal shape variants
  - [ ] monogram / icon center
  - [ ] wax color picker
- [ ] hero asset: rubber stamp
  - [ ] stamp shape
  - [ ] custom text
  - [ ] ink color and stamp angle
