# distressionaries

a figma plugin for generating fully customizable digital journaling assets. vectors, ephemera, trinkets, and composition tools — everything you'd otherwise source piece by piece from pinterest, in a single plugin.

## what it does

opens a screen wizard flow where you select what you want to make, configure it step by step, preview it, and place it on your canvas as fully editable vectors.

### asset categories

**paper ephemera**
- postcards (front / back / both)
- envelopes
- receipts
- photo strips
- movie tickets
- boarding passes

**stamps and seals**
- wax seals
- rubber stamps
- linocut / hand-cut stamps

**attachment and binding**
- washi tape
- metal clips, paper clips, wooden clips
- corkboard pins
- buttons

**photography**
- polaroids (toggle: clipped / pinned / plain)

**signage**
- wooden direction signs (strung up / grounded / free)
- wooden signboards
- street signs

**animated media**
- cassette tapes
- vinyl records
- cds

**composition tools**
- string polaroids with clips onto bulletin boards
- multi-select stringing and arrangement

## rendering modes

every asset can be generated in one of three visual modes:

| mode | description |
|---|---|
| doodle | hand-drawn illustration aesthetic. sub-toggle: filled or lineart. |
| general | clean, precise vectors. no texture, no wobble. |
| textured | clean vectors with a user-selected texture overlay (paper grain, canvas, linen, kraft, etc.) |

## design principles

- all output is vector — fully editable paths, not rasterized
- every text field is its own labeled layer
- pixel-level control over every element
- preview before placing
- multi-step wizard flow, not a wall of controls
